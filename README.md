\# TuPrimeraPagina+Krick



\--- Descripción

Proyecto web en Django con patrón MVT.



\--- Funcionalidades



\- 3 modelos: Autor, Categoria, Post

\- formulario de alta para cada modelo

\- búsqueda de posts por título

\- herencia de plantillas



\---- Orden sugerido de prueba



1\. Entrar a "/"

2\. Crear un autor en "/autor/nuevo/"

3\. Crear una categoría en "/categoria/nueva/"

4\. Crear un post en "/post/nuevo/"

5\. Buscar por título en "/buscar/"



\--- Notas

Proyecto realizado con function-based views, ModelForms y SQLite.
